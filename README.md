# Tbilisi JS Community Site

A community website for Tbilisi JS - a JavaScript and TypeScript RU-EN speaking community in Tbilisi, Georgia. We are united by our interest in technology and progress, a love for knowledge, and a genuine wish to support each other.

The site features information about community events, talks, speakers, partners, and provides ways to get involved with the community.

## Tech Stack

Next.js App Router with RSC, SCSS for styling and pnpm as package manager

## Getting Started

### Prerequisites

- Node.js >= 20.9
- pnpm (recommended to install and enable corepack)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/tbilisi-js/community-site.git
cd tbilisi-js
```

2. Install dependencies:
```bash
pnpm install
```

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Build for Production

```bash
pnpm build
pnpm start
```

### Linting

```bash
pnpm lint
```

## Contributing

We welcome contributions! Please follow these guidelines:

1. **We recommend creating an issue first** to discuss your proposed changes
2. Fork the repository and create your branch from `main`
3. Make your changes
4. Ensure your code passes linting (`pnpm lint`)
5. Submit a pull request with a clear description of your changes

If you're unsure about something, feel free to tag the maintainers in the issue/PR or ask in the Telegram chat.

### Pull Request Process

- **All pull requests must receive approval** before being merged
- **Please tag the maintainers in the community chat** (Telegram) when submitting a PR to ensure timely review
- Keep PRs focused and reasonably sized. PRs generated solely by AI without clear refactoring will be closed
- Include a clear description of what the PR does and why
